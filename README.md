# Python3の練習リポジトリ

- Formatter: black
- Linter: flake8
- 型定義: mypy
- テスト: pytest
- version管理: pyenv
- Python3のバージョン: 3.11.6

<!-- Formatter実行 -->
```bash
black test.py
```

<!-- flakeを実行 -->
```bash
flake8 test.py
```

<!-- mypyを実行 -->
```bash
mypy test.py
```

<!-- test実行 -->
```bash
pytest test.py
```
